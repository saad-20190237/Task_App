# Task Management REST API

## Overview
This project is a backend Spring Boot REST API designed for task management. It provides a robust and scalable solution for managing tasks, allowing users to create, read, update, and delete tasks efficiently.

## Features
- **RESTful API**: Follows REST principles for resource manipulation using HTTP methods.
- **Database Operations**: Implements CRUD operations with a database for persistent task storage.
- **API Documentation**: Includes comprehensive documentation for each endpoint, making it easy for developers to understand and utilize the API.

## API Endpoints
- **GET /tasks**: Retrieve a list of all tasks.
- **POST /tasks**: Create a new task.
- **GET /tasks/{id}**: Retrieve details of a specific task by ID.
- **PUT /tasks/{id}**: Update an existing task by ID.
- **DELETE /tasks/{id}**: Delete a task by ID.

## Database
The API interacts with a relational database to manage task data. Ensure to set up the database schema appropriately based on the provided model.

## Getting Started
1. Clone the repository.
2. Set up the required environment variables.
3. Run the application using appropriate Spring Boot commands.
